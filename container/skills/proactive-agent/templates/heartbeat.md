# Heartbeat Checklist

## Blockers
- [ ] 检查 state.md 中的 Blockers
- [ ] 有阻塞事项？提醒用户

## Overdue
- [ ] 检查 Next 中 >3天的待办
- [ ] 逾期事项提醒用户

## Patterns
- [ ] 扫描 conversations/ 识别重复请求
- [ ] 3次+ 的重复 → 提议自动化

## Follow-ups
- [ ] 检查 Decisions >7天
- [ ] 询问决策结果/进展

## Proactive
- [ ] 基于 User.Goals 主动建议
- [ ] "你提到想做 X，需要帮忙吗？"

## Archive
- [ ] 已完成的任务 → 归档经验
- [ ] 更新 CLAUDE.md 学习到的规则
